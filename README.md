### Mokepon Game
>Este es el primer proyecto relativamente grande que realicé con JavaScript cuándo comencé a aprender a programar por allá del 2021, basado en el juego de monstruos de bolsillo. Dónde tú eliges tu personaje, seleccionas tus ataques y compites contra tu rival.

Esta hecho con la finalidad de practicar las **bases de la programación**, hacer uso de buenas prácticas, tales como: **Don´t repeat yourself**, el correcto **uso de clases**, **sintáxis limpia**, **manipulación del DOM**. También hace uso librerias como **Cors** y **Express js** del lado del **Backend** para lograr que la aplicación soporte el servicio **multijugador local**.
